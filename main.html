RoboCupJunior OnStage

This page lists various websites, documents, posters, videos, and other resources relevant for RoboCupJunior OnStage. It gathers everything useful in one place for everybody who wants to learn more.

General information

General information about the RoboCupJunior OnStage League can be found on the official RoboCupJunior Website.

Rules and scoresheets

The OnStage rules are managed in a separate GitHub repository. Scoresheets are about to be migrated there soon. See the readme to find out more.

The documents are available in a printable PDF format and as HTML version. Please also note the general RoboCupJunior rules.
